## ✨ See it in Action

<div class="feature-card">

```cue
package env

// 🔧 Regular environment variables
DATABASE_URL: "postgres://localhost/mydb"
API_KEY: "secret123"
DEBUG: true
PORT: 3000

// 🔐 Secret references
DB_PASSWORD: "op://Personal/database/password"
STRIPE_KEY: "gcp-secret://my-project/stripe-api-key"

// 🎯 CUE features
BASE_URL: "https://api.example.com"
API_ENDPOINT: "\(BASE_URL)/v1"  // String interpolation
```

</div>

## 💻 Get Started in Minutes

<div class="feature-card" style="background: linear-gradient(135deg, rgba(102,126,234,0.1) 0%, rgba(118,75,162,0.1) 100%);">

```bash
# 📦 Install using Nix (recommended)
nix profile install github:rawkode/cuenv

# 🦀 Or using Cargo
cargo install cuenv

# 🐚 Initialize for your shell
eval "$(cuenv init bash)"  # or zsh, fish

# ✨ Create your first CUE package
echo 'package env
MESSAGE: "Hello from cuenv!"' > env.cue

# 🎉 Watch it work!
echo $MESSAGE
```
